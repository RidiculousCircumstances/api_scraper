Нужно получить номер телефона для каждого заказа.
1) запрос списка заказов
2) для каждого заказа:
    2.1) извлечь id
    2.2) запрос детальной заказа
        2.2.1) извлечь номер телефона

Как формируем схему:

пользователь создаёт схему запроса для списка заказов.
пользователь добавляет поле "категория"

пользователь создаёт схему запроса детальной заказа
- выбрать дата схему
- настроить локальные поля
- добавить внешнее поле
- выбрать внешнюю дата схему
- выбрать поле внешней дата схемы

после этого первым выполнится запрос внешней дата схемы (получить список заказов).
после этого из каждого респонса будет извлечено поле "id".
значение будет подставлено в запрос текущей дата схемы.
